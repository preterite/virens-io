<section class="index-lede">
  <h1>Research Workflow for Humanities Scholars</h1>
  <p class="subtitle">Open-source macOS framework — from source capture to publication</p>
  <div class="deck">
    <p>VIRENS integrates Obsidian, Bookends, DEVONthink, and macOS automation tools into a unified research environment. The system is modular — start with what you need, add components as your practice develops — and opinionated about how scholarly knowledge work should flow.</p>
    <p>Built on the <a href="https://verdantinquiry.org">Verdant Inquiry</a> methodology, VIRENS treats the organization of research materials as a methodological commitment, not a neutral technical choice.</p>
  </div>
</section>

<section class="quick-start">
  <h2>Quick Start</h2>
  <p>VIRENS installs as a structured Obsidian vault with three tiers — Starter (core structure only), Researcher (citations, reading pipeline, automation), and Rockstar (full integration suite). Pick the tier that matches where you are now; you can upgrade later.</p>
  <p>Requires macOS 12+, Homebrew, Git, and Python 3.9+. See the <a href="/documentation/getting-started/">Getting Started</a> guide for full details.</p>
</section>
